
# CI/CD Final Project

A Python Flask microservice with a complete CI/CD pipeline.

## Project Name
ci-cd-final-project

## Description
This project demonstrates Continuous Integration using GitHub Actions
(flake8 linting + nose unit tests) and Continuous Delivery using Tekton
Pipelines on Red Hat OpenShift.
